# generateur-de-mot-de-passe

Ce code sert aux utilisateurs de créer un ou plusieurs mots de passes ALEATOIRES sans utiliser leur identité ni des mots personnels quelconques . 

J'ai ajouté récemment une nouvelle fonctionnalité, qui permet à l'utilisateur de taper une/des quelconque(s) mot(s) en les séparant par une éspace puis le code vas ajouter des chiffres aléatoires et aussi demandé à l'utilisateur si il/elle veut une spécial charactère.

Pour l'exécuter : python generateur_de_mdp.py
